07 · Build Your Own
Custom Connector
REST + Webhooks
Any service with an API can connect to BlackRoad. Define your schema, map your events, and let Cece handle the rest.
POST /api/integrations/connect
→ returns connector_id + webhook_url
MCP Server
Model Context Protocol
Build MCP servers that plug directly into BlackRoad agents. Tools you define become capabilities any agent can discover and invoke.
mcp_servers: [{ url: "https://your.mcp/sse" }]
→ auto-registered in agent capability registry
