Sovereign API Reference

Connection

The Sovereign API runs on your hardware. Start it with:

npx tsx src/sovereign/cli.ts serve # Or after build: node dist/sovereign/cli.js serve

Endpoints

GET / API info and endpoint list
GET /health Health check with memory stats

Memory

POST /memory/remember Store a memory entry
GET /memory/recall?q=query Search memories
GET /memory/all List all memories
GET /memory/stats Memory statistics
GET /memory/verify Verify hash chain integrity
POST /memory/leak Create cross-agent memory leak
GET /memory/leaks List all memory leaks
GET /memory/export Export full memory bundle
GET /memory/:id Get memory by ID

Keys

POST /keys/create Generate sovereign key
POST /keys/pair Generate key pair
GET /keys/list List all keys (masked)
POST /keys/validate Validate a key
POST /keys/revoke Revoke a key
POST /keys/derive Derive agent key from master

Sessions

POST /session/start Start memory session
POST /session/end End memory session

Example: Store & Recall

# Store a memory curl -X POST http://127.0.0.1:8421/memory/remember \ -H "Content-Type: application/json" \ -d '{ "content": "User prefers dark mode and golden ratio spacing", "type": "fact", "agent": "LUCIDIA", "tags": ["preferences", "ui"], "confidence": 0.95 }' # Search memories curl "http://127.0.0.1:8421/memory/recall?q=dark+mode&agent=LUCIDIA" # Create a memory leak between agents curl -X POST http://127.0.0.1:8421/memory/leak \ -H "Content-Type: application/json" \ -d '{ "source": "LUCIDIA", "target": "OCTAVIA", "content": "Shared context about deployment strategy" }' # Generate a sovereign PAT curl -X POST http://127.0.0.1:8421/keys/create \ -H "Content-Type: application/json" \ -d '{"purpose": "pat", "scopes": ["read", "write"]}' # Verify chain integrity curl http://127.0.0.1:8421/memory/verify
