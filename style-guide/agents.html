05 · Display Rules
DO
Use the assigned agent color on dots, rings, bars, and borders only
DO
Agent names are always uppercase — LUCIDIA, ALICE, OCTAVIA, PRISM, ECHO, CIPHER
DO
Agent name text is always #fff — never use the agent's accent color on text
DON'T
Never color an agent's name with their accent — colors are for shapes, not words
DON'T
Never mix two agents' colors in a single indicator — one dot, one color
DON'T
Never use gradients for individual agent identity — gradients represent the collective
